# Flipkart Mobile Scraper

A Python-based web scraping project that extracts mobile phone information from Flipkart and stores the data in an Excel file for further analysis.

## Features
- Scrapes mobile phone listings from Flipkart
- Extracts:
  - Mobile Name
  - Price
  - Product Highlights
  - Image URL
- Stores data in a structured Pandas DataFrame
- Exports results to Excel (`.xlsx`)

## Technologies Used
- Python
- Requests
- BeautifulSoup (bs4)
- Pandas
- OpenPyXL
